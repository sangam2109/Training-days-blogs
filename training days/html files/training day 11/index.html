2. Start Learning React Js from Course

Start Learning React js from Code With Mosh Course recommended ans provided by mentor.

  • Learnt How to make First Prooject using CRA and Vite
    1. Install Node.js:
      • Make sure you have Node.js installed on your machine. You can download it from the official website: https://nodejs.org
    2. Install Vite:
      • Open your command prompt or terminal.
      • Run the following command to install Vite globally:
        • npm install -g create-vite
    3. Create a new Vite project:
      • Navigate to the directory where you want to create your project.
      • Run the following command to create a new Vite project:
        • create-vite my-react-app --template react
      • Replace my-react-app with the desired name for your project.
    4. Navigate into the project directory:
      • Change your current directory to the newly created project:
        • cd my-react-app
    5. Install project dependencies:
      • Run the following command to install the project dependencies:
        • npm install
    6. Start the development server:
      • Run the following command to start the development server:
        • npm run dev

    Now you have a new React.js project set up using Vite. You can start writing your documentation and build your HTML pages within the project structure. The development server will automatically reload the page whenever you make changes to your code.

    You can access your React application in the browser at http://localhost:3000.

    Feel free to customize your project and explore the Vite documentation for more advanced configurations and features: https://vitejs.dev

  • Learn about Structure and some important concepts used under react js. (like props , useeffect , eventhandlers) etc
